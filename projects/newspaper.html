EIT Digital Newspaper
Demo View on Github

This web application was the result of a User Interface Programming course at Universidad Politecnica de Madrid. The main goal of the project was to design and develop a newspaper application for three different platforms: web, mobile, and desktop.

The application should be able to perform typical "publish and manage" functionalities, common in many web applications. On the one hand, users can see the main headlines, read complete news, filter them by category and log in and out from the application service. On the other hand, administrators can publish, edit, and delete news.

The main technology used was Node.JS. For the user interface, HTML, CSS, and Javascript were used. Also, the application connects to an API to retrieve news and login information from the server.
