VideoCorn

VideoCorn was the result of a programming project from the Design and Development of Web Applications course at Universidad Polit√©cnica de Madrid. This project aims to implement both the front and back end of an online video platform with two types of roles: users and administrators.

On the one hand, users can log into the platform, look up and watch movies. For copyright reasons, a trailer will be displayed instead of the actual film.

On the other hand, administrators will be able to manage both users and movies in the database. To add new titles to the catalog, The Movie Database API will be utilized. Administrators will provide the title and trailer and the application will retrieve the rest of the information from the API automatically.

Regarding the implementation, Django, HTML, and Bootstrap were used for the project.
